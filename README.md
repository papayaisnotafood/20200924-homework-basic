# 20200924 作業

##  作業 1：記帳程式 (expense-app.cpp)
寫一個簡易記帳程式，可以計算消費的總額。

### 輸入範例
第一個輸入為 `count` 筆資料
接者輸入 `count` 行金額
```
3
100
50
75
```

### 輸出範例
顯示最後總金額
```
總共 225 元
```
